Lesson 3

Types of Malware – Part 1

What is Malware?

Malware is harmful software created to damage devices, steal data, spy on users, or disrupt normal computer operations. It can enter systems through unsafe downloads, infected email attachments, fake websites, or vulnerable software.

Understanding malware is essential because it is one of the most common cyber threats. Different types of malware behave in different ways, but all of them can cause serious harm if not detected early.

Main Types of Malware

Malware comes in several forms, and each type is designed for a different purpose such as spreading quickly, damaging files, or stealing information.

Basic Malware Categories:

  • Viruses
    • Attach to files or programs
    • Spread when the infected file is opened
    • May corrupt or delete data
  • Worms
    • Spread across networks automatically
    • Do not need user action
    • Can overload systems quickly
  • Trojans
    • Appear as legitimate software
    • Trick users into installing them
    • Can open a backdoor for attackers

Malware Comparison

Type Description Potential Impact
Virus Malware attached to a host file or program. File damage and data corruption
Worm Self-replicating malware that spreads through networks. Network congestion and rapid spread
Trojan Malicious program disguised as useful software. Unauthorized access and data theft
Payload The harmful action delivered after infection. System damage or spying
Backdoor A hidden entry point attackers use after infection. Long-term compromise
