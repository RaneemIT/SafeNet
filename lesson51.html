Lesson 5

Encryption – Part 1

What is Encryption?

Encryption is the process of converting readable data into an unreadable form so that only authorized users can access it. It is one of the most important methods used to protect information during storage and transmission.

When data is encrypted, it becomes much harder for attackers to understand or steal it. Encryption is widely used in websites, messaging apps, banking systems, and cloud services to keep personal and business information secure.

Basic Encryption Concepts

Encryption depends on a few key concepts that help transform readable data into protected data and then restore it when needed. These concepts form the foundation of secure digital communication.

Core Encryption Elements:

  • Plaintext
    • Original readable data
    • Can be exposed if unprotected
    • Needs encryption before transfer
  • Ciphertext
    • Encrypted unreadable data
    • Requires a key to restore
    • Safer for transmission and storage
  • Key
    • Used to encrypt data
    • Used to decrypt data
    • Must be protected carefully

Encryption Terms Comparison

Term Description Main Purpose
Plaintext The original readable message or file before encryption. Starting form of data
Ciphertext The encrypted unreadable version of the message. Protects information from unauthorized reading
Symmetric Key The same key is used for encryption and decryption. Faster encryption process
Asymmetric Key Uses a public key and a private key pair. Secure sharing and communication
Decryption The process of restoring encrypted data to readable form. Allows authorized access to data
